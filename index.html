HTML5

HTML (Hyper Text Markup Language) is the standard markup language for documents designed to be displayed in a web browser. It can be assisted by technologies such as Cascading Style Sheets and scripting languages such as JavaScript.

CSS3

CSS (Cascading Style Sheets) is the language used to format the layout of Web pages. They can be used to define text styles, table sizes, and other aspects of Web pages that previously could only be defined in a page's HTML

JavaScript

JavaScript is a dynamic computer programming language. It is lightweight and most commonly used as a part of web pages, whose implementations allow client-side script to interact with the user and make dynamic pages. It is an interpreted programming language with object-oriented capabilities.

Node.js

Node.js is a platform built on Chrome's JavaScript runtime for easily building fast and scalable network applications.

Git

Git is a distributed version-control system for tracking changes in source code during software development. It is designed for coordinating work among programmers.

GitHub

GitHub is a web-based version-control and collaboration platform for software developers.

Getting To Know Me

About Me

From the very beginning of my career, as a Radiologic Technologist, I found myself fascinated by the software and equipment. After dedicating 18 years to mastering the use of various types of medical imaging software, I found myself becoming more interested in how to build that same software.

Curiosity has always been my friend, so I decided to start doing some reading. Little did I know the gift that was waiting to be unwrapped. Software Engineering has become a growing passion and has fed a hunger that I’ve always had. The vast amount of knowledge and forever evolving technologies allow for limitless possibilities for learning, growth and success.

My Interests

Healthcare Technology has always been an interest of mine. It has been since the first day I started volunteering at my local hospital and always will be.

Snowboarding is another passion I have. I spent years living in Lake Tahoe exploring the Sierra Nevada Mountains and competed in multiple snowboarding competitions.

Mammography and promoting Breast Cancer Awareness are my 2 most important interests. Most of my professional growth was concentrated on Mammography and X-ray, but Mammography was where I belonged. Having the opportunity to work with and educate women on their breast health is the most rewarding and fulfilling work I have ever done.
